Pexip Video Connect

Congrats! The integration widget was correctly installed. However, in order to use this integration, you will need to perform some additional configuration steps in Genesys and in Pexip Infinity. To help you with this task, we have two resources that will be very handy for you.

Genesys Documentation contains all the information about how to configure this integration, and if you have any questions, feel free to contact us through the Community Portal.

Genesys Documentation Community Portal

To help you with the configuration, here is a list of the steps that you need to follow to have the integration up and running. You can find the extended explanation of each step in the documentation.

Genesys configuration steps:

  • Add phone number for the inbound trunk.
  • Create the SIP trunk.
  • Enable Ciphers and Media Settings for the SIP trunk.
  • Create/identify a call queue.
  • Create/identify a call flow.
  • Assign the integration to a group of users.

Pexip Infinity configuration steps:

  • Configure the call routing to the SIP trunk.
  • Configure the local policy.

Additionally, you can perform other optional configuration in Pexip infinity, such as media playback services, customized branding and themes.
